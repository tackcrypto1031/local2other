🔐 SSH 設定指南

在遠端電腦安裝 OpenSSH Server

以管理員身份開啟 PowerShell，執行以下指令：

1. 安裝 OpenSSH Server
Add-WindowsCapability -Online -Name OpenSSH.Server~~~~0.0.1.0
2. 啟動 SSH 服務
Start-Service sshd
3. 設定開機自動啟動
Set-Service -Name sshd -StartupType 'Automatic'
4. 確認防火牆規則（通常安裝時會自動設定）
Get-NetFirewallRule -Name *ssh*
💡 提示：若防火牆規則不存在，執行：
New-NetFirewallRule -Name sshd -DisplayName 'OpenSSH Server (sshd)' -Enabled True -Direction Inbound -Protocol TCP -Action Allow -LocalPort 22

在本機生成 SSH 金鑰

設定金鑰後，可實現免密碼自動傳輸。

檢查金鑰狀態...
手動生成指令（在 PowerShell 執行）
ssh-keygen -t rsa -b 4096

將公鑰部署到遠端電腦

複製下方公鑰內容，貼到遠端電腦的 C:\Users\使用者\.ssh\authorized_keys 檔案中。

載入公鑰...
⚠️ 注意：
  • 若 .ssh 資料夾不存在，請手動建立
  • 若 authorized_keys 檔案不存在，請建立新檔案
  • 每行放一個公鑰，可以有多個公鑰
使用指令部署（進階）
一鍵部署到遠端（需要輸入密碼一次）
type $env:USERPROFILE\.ssh\id_rsa.pub | ssh 使用者@IP "mkdir -Force ~\.ssh; Add-Content ~\.ssh\authorized_keys"
